[[appendix]]

-> Glossary of Rust's syntax
# Operators
-> Behave like functions. Ex: `+ - / ? !` etc.
# Non-operators
-> Ex. `'ident, u8, i32, r"...", b"...", '...', |...| expr, !, ,_, ::`

-> `#[meta]` outer attribute
-> `#![meta]` inner atribute
-> `$ident` macro substitution
-> `$ident:kind` macro metavariable
-> `$(...)...` macro repetition
-> `ident!(...), ident!{...}, ident![...]` macro invocation
