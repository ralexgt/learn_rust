[[appendix]]

-> For formatting standars - `rustfmt` and `cargo-fmt` that uses rustfmt to format cargo projects by the community standards
-> `cargo fmt` formats any Cargo project

-> `cargo fix` used to fix **warnings** with clear fixes and it would be what the programmer would have done as well.
-> Also used when you switch Rust editions

-> `cargo clippy` is a collection of lints to analyze your code and catch common mistakes + improve Rust code

-> `rust-analyzer` for IDE integration
