[[rust by example]] [[introduction]]
-> `rustc` compiler
-> `//` line comment
-> `/**/` block comment
-> `///` documentation for the following item
-> `//!` doccumentation for the enclosing item
-> `cargo doc` to build documentation in `target/doc` and `cargo doc --open` to automatically open it in the browser
-> `cargo test` runs all test, including documentation tests. `cargo test --doc` runs only the documentation tests

-> Formatting text
-> Printing is handled by a handful of `macros` defined in `std::fmt`
-> `format!` `print!` `println!` `eprint!` `eprintln!`
-> `fmt::Debug` and `fmt::Display` govern the display of text, there are other too
-> `#[derive(Debug)]`
-> `std::fmt::Display`
